什么是链接小卡片

实际上叫badge，翻译过来是徽标的意思。因为我每次用这个都是用作链接，而它长得有点像小卡片，所以贴心地起名叫“链接小卡片”。

使用方法

链接小卡片本质上是借助 这个网站来实现的图片。所以会用到 Markdown 图片语法，如果想做成链接，还需用到 Markdown 链接语法。

所有参数值未加特殊说明均不可省略！

由于是通过其他网站实现显示图片，所以要想正常显示，必须确保有网。
